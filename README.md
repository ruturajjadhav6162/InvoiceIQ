# InvoiceIQ

To run Backend 
1) ```cd backend```
2)```uvicorn invoiceIQ:app --reload```

To run streamlit
1) ``` streamlit run .\backend\streamlit.py ```


This are the env required to run the backend successfully 

`GOOGLE_API_KEY = "" 
GROQ_API_KEY= ""`

To run Frontend

1) ```npm install ```
2) ``` npm run dev```
